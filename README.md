# Computational Analysis of Single Cell Data
This repository contains the materials for the course on Computational Analysis of Single Cell Data at Heidelberg university for the DSCB major.

## Preparations
Please bring a laptop with R (version 4.5) and RStudio installed. Before the course, make sure to install all required packages listed in [this](installation_instructions.R) script so that we can start directly with the hands-on exercises.
If you encounter any issues during installation, don’t worry — we will help you resolve them at the beginning of the course.
